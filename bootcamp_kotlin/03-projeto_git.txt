Projeto Open Source
é um projeto de código aberto, livre para ser utilizado,
modificado e distribuido. Exemplos.:
Linux, sob a licença GNU General Public License (GPL)
Roadmaps DIO, sob a licença MIT License (MIT)

Formas de contribuir em um Projeto Open Source
- Relatar algum problema ou bug
- Sugerir melhorias ou adição de novos recursos
- Escrever ou atualizar a documentação
- Contribuir com o código
- Divulgar e/ou doar

git remote add upstream url
faz referência ao repositório original e mantém o repositório
atualizado como ele

git pull upstream main
baixa as alterações do repositório remoto original
